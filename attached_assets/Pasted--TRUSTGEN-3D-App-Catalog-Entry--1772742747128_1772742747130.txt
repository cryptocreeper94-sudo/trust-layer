═══ Marketing Description ═══
TrustGen is a browser-based, AI-powered 3D creation platform where 
every asset is hallmarked on Trust Layer's blockchain. Create stunning 
3D scenes with professional-grade tools, generate models from text 
descriptions using AI, animate with keyframe timelines, and export 
production-ready assets — all without downloading any software.

TrustGen combines the power of Three.js real-time rendering with 
Meshy.ai's AI model generation and Trust Layer's immutable provenance 
verification. Every 3D creation receives a unique hallmark on the 
blockchain, providing cryptographic proof of authorship, timestamp, 
and authenticity. Creators earn Signal (SIG) rewards for platform 
activity through the Trust Layer ecosystem.

The platform features a cinematic Explore page with rotating hero 
imagery, a PIN-secured Developer Portal for system administration, 
an AI-driven blog for SEO content, ecosystem-wide Signal Chat for 
real-time support, and comprehensive legal pages. TrustGen is fully 
mobile-optimized and integrates seamlessly with the broader Trust Layer 
ecosystem of 12+ connected apps and services.

═══ Key Features ═══
• 3D Creation Studio — primitives, materials, lighting, real-time preview
